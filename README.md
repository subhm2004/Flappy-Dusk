<h1 align="center">Flappy Dusk</h1>

<p align="center">
  A pretty, playable <b>3D flappy-bird</b> game — coins, power-ups, rare keys,
  daily missions, levels, and a bird shop.<br>
  Built with <b>Next.js</b>, <b>TypeScript</b>, and <b>Three.js</b>.
</p>
